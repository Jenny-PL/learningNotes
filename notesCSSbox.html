CSS: part 2

Position

  • Static: default: stays where it's at. Adding values to top/bottom/L/R have no effect.
  • Relative: moves relative to it's original space in the document. Holds its original space available
  • Absolute: element is removed from the normal flow and no space is created for the element. It is positioned relative to it's closest positioned (aka not set to 'static') ansestor. If not element is positioned, it will be relative to the body.
  • Fixed: Removed from flow; no space created for it. Positioned relative to initial containing block (ie-body). It stays here. Often used for Nav bars.
  • sticky: Starts at original spot, but when scroll down, ends up positioned relative to initial to containing block. **For some reason, it's not working!!**

Box model

Borders

Width, style, color, border radius (to round corners or make a circle). Can specify separately or all at once.
Shorthand: border: 30px solid grey with width style color
Then can specificy border-left: width style color

Padding: Spacing Inside an element, between content and borders

Shorthand:
  • Padding: all sides
  • Padding: vertical horizontal
  • Padding: top horizontal bottom
  • Padding: top right bottom left (clock wise)

Margin: Spacing outside of the element

Display

  • Inline: Width and height are ignored. Margin & padding push elements away horizontally but not vertically
  • Block: Break the flow, place subsequent elements on the next line. width, height, padding, margin are respected
  • Inline-block: Behaves like an inline element, except width, height, padding, margin are respected
  • None: Element is present on the page, but not seen.
