Options for CSS location:

  1. Inline. within actual html tag, using style= "color:red;"
  2. Internal: Within style tag placed inside head element.

General Principals

Order matters!

The last declared CSS style for an element will be the style that trumps... in the case that the same element is given two conflicting colors, for example

  2. Class, attribute, pseudo-class selectors

Selectors

  1. Elements

    (ie- h1, p, li, a, span, div, etc..)

  2. Class (label multiple elements together to style the same).


    TIP: hold option to get multiple cursers and label them all at once.
    To utilize: .name { color:----;}

  3. IDs (# that are referred to in the html; ie- to style one specific button

    To utilize: #name { color:----;}

  4. Descendent selector

    Use a space between names of elements to include all of a specific element nested within another (ie- li a {color: blue}; This means all anchor tags within a list element)

  5. Adjacent selectors: h2 + button {color: blue;}

    This means any button that comes after an h2 will be blue

  6. Direct Descendent/child: footer >a {color: purple;}

    This means that any anchor tag that is a direct descendent (not right after necessarily, but just one level down... not nested within something else) of an footer will be purple

  7. Attribute selector: input[type="text"] {color: blue;}

    This specifies a certain attribute of an element for styling.

  8. Psuedo class a:hover {background-color: light purple;}

    • Hover: change size, background color, text color, etc when a button, link, text, picture is hoevered over.
    • Active: Change an attribute when an element is activated (ie- anchor tag is clicked on)
    • nth-of-type(3n): This will make every 3rd of an element affected. Alternately, nth-of-type(3) would make only the 3rd of an element be affected. nth-of-type(2n+1) means every other, starting with 1 (1st of an element), while nth-of-type(2n) means every other, so it will start with the 2nd element.

  9. Psuedo element h2::first letter { font-size: 30px;}

    Can choose first letter, first line, selected element, before, after, etc

Color (a text attribute)

element {color: #11999e;} p {color: rgb (_,_,_);}

  • Colors are a mix of red, green, blue on scale from 0 to 255. color: rgb (255,255,255) is white. color: rgb (0,0,0) is black. Use a color picker to find color mixes and complementary colors/palettes.
  • hexadecimal: 0123456789ABCDEF (aka 0 to 15. F=15, making a two-digit # into 1 digit) 0 to ff is equal to 0 to 255. uses red-blue-green mixes.

Alpha & opacity

Alpha: used with rgba (0,250,140,0.4) as a fourth digit, that gives a specifically selected item a level of oppacity. 1= not at (Can aslo be used with hexadecimal, adding two extra digits/letters, 0 to F, at end to specify opacity)
Opacity: relates to the entire element, not just one selected item (ie- background color, text color, images, etc).

Font Weight & Size

  • Weight: 400=normal, 700=bold (range of 0-1000, inclusive)
  • Size: font-size:__;
    • Relative (small, medium large, em, rem, etc.):
    • Em: Realtive to the element's default size (this is not a constant default within the document). Ie- h1, h2, p all have different default sizes. So 1em may be a different value (#px) for each element.
    • Rem (root em): relative size, compared with a constant default within the document (set how?) (ie- 16px =1em, so 1.5em is 24px.)
    • Absolute (px, etc...)

Font Family

  • Font-family: ____;
  • Font stack: lists font options. If one font is not available, the next best choice is listed next, separated by commas. (Ex: Font-family: Avant garde, impact, san-serif ;)
    • San-serif above is a generic family name, thus requesting any font within that family
  • Commonly supported fonts on Macs and Windows
  • Google fonts : Can download fonts to use in project. Copy code and paste into 'head' of HTML doc
  • NOTE: Use quotations if there is a space or a character other than - within the name. Ie: "Gills Sans", sans-serif, cursive
