Python

Data types

Date types: bool (True, False), int (integer), float (# with decimal), str (strings), range, list (similar to arrays- but not a fixed size), tuple (example: x,y or coordinates) , dict (dictionary- can store any key with any value), set (a collection of values without duplicates)

Random tid bits

  • Indentation matters
  • Capitalize True, False bools
  • parameters are variables used to refer to pieces of data provided as input to a function (defined in the function signature)
  • arguments are those pieces of data passed to the function when it gets called (specific values given to the function when it's invoked)
  • docstring: multi-line comment that can be within a function, module, etc, to give more info (tells input and output)
    • ''' 3 single or double quotations mark start and end of docstring
    • Must be the first line of the function
    • A docstring can be called up using:
    • print(function_name.__doc__)
    • two underscores before and after doc

Strings

  • Concatenation
  • Slicing:
    • L to R: 0 to end length (ie- Python: 0 to 5, where word length is 6)
      • name = tommy
        name [2:4] = 'mm'
        The range is from first spot to last, not inclusive
    • R to L: -1 to first letter (ie- Python is -1 to -6, from N to P)
    • This is because -0 is also 0, so the last letter must instead be -1.
    • string[i:j:k] inlcudes range from i to k, not inclusive, and excludes j.
      • name = timothy
      • name[0:2:4] = 'tio'
    • len() will return string length. (ie- len(python) = 6)
    • Indexing: string[i], will return one selected character from string
  • String interpolation
    is subbing a value from a variable into a string
    • print(f"{state_name1.capitalize()} is a nice place to live")
      state_name is the variable
      capitablize() is the method
      method is a function for a specific class.
    • f is for format
    • The value to be interpolated is placed within curly braces {},

Loops

  • While loops
    • i = 0
      while i < 3:
      print("hello, world")
      i += 1
  • For loops
    • for i in range(3):
      print("hello, world")
    • This will print 3x, as range is 0,1,2
  • Do-While loops

Functions

Take an argument as input, and return a value as output.

Common functions

  • type takes an argument, and returns the data type as output.
  • bool takes data, returns the boolean True or False.
  • str converts data to the string class.
  • int converts data to the integer class.
  • float converts data to the float class
  • abs returns the absolute value of the value
  • round returns the value rounded to the nearest integer
  • input accepts user input

Making (aka Defining) Functions

  • def greet_jenny():
    print("Hi, Jenny")
  • To use the function, you call (or invoke) it:
    greet_jenny()

Modules (often called libraries)

  • Examples: math, random
  • You must import the module
    import module_name
  • To use the functions:
    module_name.function_name
    • import random
    • random.randint(a,b)
      • Returns a random integer N such that a <= N <= b
    • random.random()
      • returns a random floating point number in the range [0.0, 1.0).
