Transitions

Property name, duration, timing function, delay

Identify properity, then list duration of transition, use a timing function if desired (ie- ease in, ease out, steps, cubic-bezier), then delay to start transition if desired. Often used with HOVER I.e.: transition: background-color 1s ease-in, border-radius 2s;

Backgrounds

Unsplash: A website with great free images
Attributes: size, color, position, etc.
  • Contain: Scales the image as large as possible within its container without cropping or stretching the image. Then image will repeat unless set to "no-repeat". This is background-size attribute.
  • Cover: Scales the image as large as possible to fill the container, stretching if necessary.
